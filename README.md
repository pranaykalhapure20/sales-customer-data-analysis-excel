# Sales and Customer Data Analysis using MS Excel

## Project Overview

This project focuses on analyzing sales and customer data using Microsoft Excel to identify business trends, customer behavior, and key performance indicators. The data was cleaned, organized, and analyzed using various Excel features to generate meaningful insights and support data-driven decision-making.

## Objectives

* Organize and manage sales and customer data efficiently.
* Analyze sales performance and customer trends.
* Create interactive dashboards for data visualization.
* Generate insights to support business decisions.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Excel Formulas and Functions
* Data Sorting and Filtering
* Conditional Formatting
* Dashboard Design

## Key Features

* Data cleaning and preprocessing
* Sales performance analysis
* Customer trend analysis
* Interactive dashboard creation
* Visual reports using charts and graphs
* KPI tracking and business insights

## Project Workflow

1. Collected and organized raw sales and customer data.
2. Cleaned and prepared the dataset for analysis.
3. Applied Excel formulas, sorting, and filtering techniques.
4. Created Pivot Tables and Pivot Charts for analysis.
5. Designed an interactive dashboard to visualize key metrics.
6. Generated reports and insights for decision-making.
